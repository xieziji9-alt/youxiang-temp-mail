  遇到问题？
═══════════════════════════════════════════════════════════

问题1: npm 安装失败（权限错误）
解决方案：
  - 以管理员身份运行命令提示符
  - 或使用 yarn: npm install -g yarn && yarn install
  - 详细方案查看: INSTALL.md

问题2: 端口 3000 被占用
解决方案：
  - 修改端口: set PORT=3001 && npm run dev
  - 或关闭占用 3000 端口的程序

问题3: 收不到邮件
说明：
  - 需要配置邮件服务器（Postfix）
  - 需要配置域名 DNS 记录
  - 详细配置查看: SETUP.md 和 DOMAIN_SETUP.md
