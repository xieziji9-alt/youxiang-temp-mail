═══════════════════════════════════════════════════════════
  如何添加更多三级域名
═══════════════════════════════════════════════════════════

编辑 app/page.tsx 文件：

找到这部分代码：
────────────────────────────────
const SUBDOMAINS = [
  'cheyu',
  'yumail',
  'tempmail',
  'mail'
]
────────────────────────────────

添加你想要的三级域名：
────────────────────────────────
const SUBDOMAINS = [
  'cheyu',
  'yumail',
  'tempmail',
  'mail',
  'test',      // 添加 test.xieziji.shop
  'demo',      // 添加 demo.xieziji.shop
  'temp'       // 添加 temp.xieziji.shop
]
────────────────────────────────

保存后重启服务器即可！

═══════════════════════════════════════════════════════════
  DNS 配置说明
═══════════════════════════════════════════════════════════

✅ 优势：只需配置一次通配符记录

在域名管理后台添加：
────────────────────────────────
xieziji.shop        A    你的服务器IP
*.xieziji.shop      A    你的服务器IP
────────────────────────────────

这样配置后，所有三级域名都会自动生效：
  ✅ cheyu.xieziji.shop
  ✅ yumail.xieziji.shop
  ✅ tempmail.xieziji.shop
  ✅ mail.xieziji.shop
  ✅ 任何你添加的三级域名

无需为每个三级域名单独配置！

═══════════════════════════════════════════════════════════

Q: 为什么选择三级域名而不是多个域名？
A: 
  - 只需配置一次 DNS（通配符记录）
  - 更容易管理和扩展
  - 所有邮箱都在同一个主域名下
  - 更专业和统一

Q: 可以同时支持多个主域名吗？
A: 
  可以！如果需要，可以修改代码支持：
  - xieziji.shop 的三级域名
  - otherdomain.com 的三级域名
  
  但当前版本专注于 xieziji.shop

Q: 如何修改默认选中的三级域名？
A: 
  编辑 app/page.tsx，找到：
  const [selectedSubdomain, setSelectedSubdomain] = useState(SUBDOMAINS[0])
  
  修改 [0] 为其他索引：
  [0] = cheyu
  [1] = yumail
  [2] = tempmail
  [3] = mail

Q: 三级域名的邮件会分开存储吗？
A: 
  不会，所有邮箱的邮件都存储在一起。
  每个邮箱地址是独立的，互不影响。
