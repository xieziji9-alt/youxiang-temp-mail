# xieziji.shop 三级域名使用指南

## 🎯 概述

平台现在支持 **xieziji.shop** 的多个三级域名，用户可以选择不同的三级域名来生成临时邮箱。

## ✅ 支持的三级域名

| 三级域名 | 完整域名 | 示例邮箱 |
|---------|---------|---------|
| cheyu | cheyu.xieziji.shop | abc123@cheyu.xieziji.shop |
| yumail | yumail.xieziji.shop | def456@yumail.xieziji.shop |
| tempmail | tempmail.xieziji.shop | ghi789@tempmail.xieziji.shop |
| mail | mail.xieziji.shop | jkl012@mail.xieziji.shop |

## 🔧 如何添加新的三级域名

### 步骤 1: 编辑代码

打开 `app/page.tsx` 文件，找到：

```typescript
const SUBDOMAINS = [
  'cheyu',
  'yumail',
  'tempmail',
  'mail'
]
```

添加你想要的三级域名：

```typescript
const SUBDOMAINS = [
  'cheyu',
  'yumail',
  'tempmail',
  'mail',
  'test',      // 新增：test.xieziji.shop
  'demo',      // 新增：demo.xieziji.shop
  'temp'       // 新增：temp.xieziji.shop
]
```

### 步骤 2: 重启服务器

```bash
# 停止当前服务器（Ctrl+C）
# 重新启动
npm run dev
```

### 步骤 3: 测试

访问 http://localhost:3000，你会看到新增的三级域名选项。

## 🌐 DNS 配置

### 优势：一次配置，永久生效

只需在域名管理后台配置一次通配符记录：

```
记录类型: A
主机记录: *
记录值: 你的服务器IP
TTL: 600
```

完整配置：

```
xieziji.shop        A       123.45.67.89
*.xieziji.shop      A       123.45.67.89
xieziji.shop        MX      10 mail.xieziji.shop
```

### 自动生效的域名

配置通配符后，以下所有域名都会自动指向你的服务器：

- ✅ cheyu.xieziji.shop
- ✅ yumail.xieziji.shop
- ✅ tempmail.xieziji.shop
- ✅ mail.xieziji.shop
- ✅ test.xieziji.shop（如果你添加了）
- ✅ demo.xieziji.shop（如果你添加了）
- ✅ 任何你在代码中添加的三级域名

**无需为每个三级域名单独配置 DNS！**

## 🔄 与之前版本的区别

### 之前（多个独立域名）

```typescript
const DOMAINS = [
  '886178.xyz',
  '88617.xyz',
  '886178.shop',
  'xieziji.shop'
]
```

**问题：**
- 需要管理多个域名
- 每个域名都要单独配置 DNS
- 成本较高

### 现在（xieziji.shop 三级域名）

```typescript
const SUBDOMAINS = [
  'cheyu',
  'yumail',
  'tempmail',
  'mail'
]

const BASE_DOMAIN = 'xieziji.shop'
```

**优势：**
- ✅ 只需一个主域名
- ✅ DNS 配置一次即可
- ✅ 易于扩展和管理
- ✅ 成本更低
- ✅ 更专业统一

## 📊 技术实现

### 邮箱生成逻辑

```typescript
const generateEmail = () => {
  // 生成随机字符串
  const randomString = Math.random().toString(36).substring(2, 8) + 
                      Math.random().toString(36).substring(2, 8)
  
  // 组合三级域名
  const fullDomain = `${selectedSubdomain}.${BASE_DOMAIN}`
  
  // 生成完整邮箱地址
  setEmailAddress(`${randomString}@${fullDomain}`)
}
```

- 选择 `tempmail` → `qwe456rty789@tempmail.xieziji.shop`

## 🎯 最佳实践

### 1. 命名规范

三级域名建议：
- ✅ 简短易记（如：mail, temp, test）
- ✅ 有意义（如：cheyu, yumail）
- ✅ 小写字母（避免大小写混淆）
- ✅ 不使用特殊字符

### 2. 数量控制

建议三级域名数量：
- 最少：2-4 个（基本够用）
- 适中：4-8 个（满足大多数需求）
- 最多：不超过 12 个（避免选择困难）

### 3. 默认选择

设置最常用的三级域名为默认：

```typescript
// [0] 表示第一个，即 'cheyu'
const [selectedSubdomain, setSelectedSubdomain] = useState(SUBDOMAINS[0])

// 如果想默认选择 'yumail'，改为：
const [selectedSubdomain, setSelectedSubdomain] = useState(SUBDOMAINS[1])
```

## 🔐 安全建议

1. **不要使用敏感词**
   - ❌ admin.xieziji.shop
   - ❌ root.xieziji.shop
   - ✅ mail.xieziji.shop

2. **避免混淆**
   - ❌ mail 和 mai1（容易混淆）
   - ✅ mail 和 temp（清晰区分）

3. **保持一致性**
   - 所有三级域名都使用小写
   - 统一命名风格

## 📈 扩展性

### 动态加载三级域名

未来可以从数据库或配置文件加载：

```typescript
// 从 API 加载
const [subdomains, setSubdomains] = useState([])

useEffect(() => {
  fetch('/api/subdomains')
    .then(res => res.json())
    .then(data => setSubdomains(data))
}, [])
```

### 用户自定义

允许用户创建自己的三级域名：

```typescript
const createCustomSubdomain = (name: string) => {
  // 验证名称
  if (isValidSubdomain(name)) {
    // 添加到列表
    setSubdomains([...subdomains, name])
  }
}
```
