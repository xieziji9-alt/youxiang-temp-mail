========================================
   临时邮箱平台 - 快速开始
========================================

🎯 你想做什么？

【1】本地测试（在自己电脑上运行）
    → 双击运行：dev.bat
    → 然后访问：http://localhost:3000

【2】部署到网上（让所有人都能访问 xieziji.shop）
    → 第一步：双击运行 setup-github.bat
    → 第二步：双击运行 deploy.bat
    → 第三步：打开文件 一键部署指南.md
    → 按照指南完成 Cloudflare 配置

========================================

📖 详细文档

- 一键部署指南.md ⭐⭐⭐ 最重要！
- 项目文件说明.md - 了解每个文件的作用
- CLOUDFLARE_DEPLOY.md - Cloudflare 详细配置

========================================

❓ 常见问题

Q: 本地运行失败？
A: 以管理员身份运行 dev.bat

Q: 推送到 GitHub 失败？
A: 先运行 setup-github.bat 配置

Q: 域名无法访问？
A: 等待 DNS 生效（最多24小时）

========================================

💡 提示

1. 首次部署需要 15-20 分钟
2. 完全免费，不需要服务器
3. 支持自动更新，改代码后 git push 即可
