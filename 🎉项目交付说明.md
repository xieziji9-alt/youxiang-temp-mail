## 🔧 自定义修改

### 修改子域名列表

编辑 `app/page.tsx`，找到：

```typescript
const SUBDOMAINS = ['cheyu', 'yumail', 'tempmail', 'mail']
```

改为你想要的子域名：

```typescript
const SUBDOMAINS = ['mail', 'temp', 'inbox', 'box']
```

### 修改主域名

编辑 `app/page.tsx`，找到：

```typescript
const BASE_DOMAIN = 'xieziji.shop'
```

改为你的域名：

```typescript
const BASE_DOMAIN = 'yourdomain.com'
```

### 修改界面颜色

编辑 `app/page.tsx`，搜索 `blue-` 替换为其他颜色：
- `red-` 红色
- `green-` 绿色
- `purple-` 紫色
- `pink-` 粉色

### 修改邮件过期时间

编辑 `functions/api/emails.ts`，找到：

```typescript
expirationTtl: 3600 // 1小时
```

改为你想要的秒数：

```typescript
expirationTtl: 7200 // 2小时
```

## 🆘 遇到问题？

### 问题排查步骤

1. **查看文档**
   - 打开 `一键部署指南.md`
   - 查看"常见问题"部分

2. **检查日志**
   - Cloudflare 控制台 → 查看构建日志
   - 浏览器 F12 → 查看 Console 错误

3. **验证配置**
   - GitHub 仓库是否创建？
   - Cloudflare Pages 是否部署成功？
   - 域名 DNS 是否生效？
   - Email Routing 是否启用？

### 常见问题快速解决

| 问题 | 解决方案 |
|------|----------|
| npm 安装失败 | 以管理员身份运行 |
| Git 推送失败 | 运行 setup-github.bat 配置认证 |
| 构建失败 | 检查 Node.js 版本是否为 18 |
| 域名无法访问 | 等待 DNS 生效（最多24小时）|
| 收不到邮件 | 检查 Email Worker 和 KV 配置 |
